# Healthcare-Data-Analysis-and-Insights
Healthcare data analysis using Excel, focusing on patient health metrics, costs, and interactive dashboard insights.

## PROJECT OVERVIEW 

This project analyzes healthcare data to understand patient health conditions, medical history, and treatment costs. It focuses on identifying patterns between factors like BMI, diabetes, age, and healthcare charges. Data cleaning, transformation, and analysis were performed to prepare the dataset for insights. Visualizations and dashboards were created to explore trends and relationships. The findings help in improving decision-making in healthcare management.

## TOOLS USED

 * Excel (Data Cleaning, Transformation, Dashboard)

 * Pivot Tables & Charts

 * Data Visualization

## STEPS FOLLOWED

### 1. Data Cleaning

 * Handled missing values and replaced invalid entries

 * Filled missing values using mean, mode, and default values

 * Standardized categorical columns like smoker, hospital tier, and city tier

### 2. Data Transformation

Split customer names into Title, First Name, and Last Name

Converted non-numeric values into proper numerical format

Created new columns: Weight Status and Diabetes Status

Calculated Age using Date of Birth

Formatted charges as currency


### 3. Data Integration

 * Merged multiple tables using Customer ID

 * Selected relevant columns for analysis

 * Created a final combined dataset for reporting Data Transformation

### 4. Data Analysis & Visualization

 * Used pivot tables for analysis

 * Created pie charts, bar charts, and line charts

 * Analyzed relationships between age, BMI, HbA1C, and charges

### 5. Dashboard Creation

 * Built an interactive dashboard in Excel

 * Added slicers for Weight Status and Diabetes Status

 * Ensured clear and user-friendly visualization of insights

## KEY INSIGHTS

 * Higher BMI and diabetes levels are associated with higher healthcare charges

 * Older patients tend to have higher medical expenses

 * Smokers show higher risk factors compared to non-smokers

 * Patients with transplants or major surgeries have higher HbA1C and costs

 * Hospital tier and location impact healthcare charges
